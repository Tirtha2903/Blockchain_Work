# Stellar Wallet

A simple web-based wallet for Stellar blockchain transactions.

## Features

- Create new Stellar accounts (funded with test XLM on testnet)
- Check account balance
- Send XLM payments to other accounts

## Getting Started

1. Install dependencies: `npm install`
2. Start the server: `npm start`
3. Open your browser and go to `http://localhost:3001`

## Note

This app uses Stellar testnet. All transactions are on testnet and use test XLM.

Keep your secret keys safe!
